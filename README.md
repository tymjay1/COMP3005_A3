Important note:
Make sure your current working directory is under the COMP3005_A3/src folder.
And make sure to change the url, user, and password in the connect() function if you want to connect with your own database.

Compile:
```
javac -cp ".:lib/postgresql-42.7.8.jar" PostgreSQLJDBCConnection.java
```
After this, a PostgreSQLJDBCConnection.class will get generated

Run:
```
java -cp ".:lib/postgresql-42.7.8.jar" PostgreSQLJDBCConnection
```
